# 恋愛モビー診断 相性表 100点版

重複なしの全136パターンです。A×BとB×Aは同じ相性として扱います。

## 判定軸
- A軸: H = Heroine / ヒロイン、S = Support / 支え役
- B軸: L = Lock / そばにいたい、F = Free / 自由もほしい
- C軸: T = Tokimeki / ときめき、A = Anshin / 安心
- D軸: O = Open / 見せたい恋、C = Close / 内にしまう恋

## スコアの目安
- 90〜100点: 最高相性
- 80〜89点: かなり良い
- 70〜79点: 良い
- 60〜69点: 普通に合う
- 50〜59点: 工夫で伸びる
- 40〜49点: ズレやすい
